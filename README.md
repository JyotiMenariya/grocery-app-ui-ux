# 🛒 FreshCart - Grocery Shopping App UI/UX Design

FreshCart is a modern and user-friendly grocery shopping mobile application designed in **Figma**. The project focuses on creating a clean, intuitive, and visually appealing shopping experience for users.

## 📌 Project Overview

This UI/UX design includes the complete user journey for an online grocery shopping application, from user authentication to product browsing, ordering, payment, and order tracking.

## ✨ Features

- 🔐 Login Screen
- 🏠 Home Screen
- 🔍 Product Search
- 🥬 Category Browsing
- 🛍️ Product Listing
- 👤 User Profile
- 📦 Order Confirmation
- 💳 Payment Method Selection
- ✅ Payment Success Screen
- ❌ Payment Failure Screen
- 🚚 Order Tracking
- 🧭 Bottom Navigation
- 🎨 Reusable UI Components
